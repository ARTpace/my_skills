# xurl：面向 WordPress 和 Shopify 的 Twitter 内容情报

`xurl` 技能将 Twitter 转化为强大的洞察引擎、权威建设者和潜在客户雷达，用于吸引 WordPress 和 Shopify 客户。

**注意：** `xurl` 处理你提供的 Twitter 内容（例如，复制的个人资料、帖子串或对话文本）。它不直接访问 Twitter API。

## 核心功能

### 1. Twitter 个人资料分析
分析个人资料以了解定位、目标受众、语调、内容主题和互动模式。

### 2. 帖子串拆解
解构病毒式或高表现力的帖子串。提取核心思想，识别钩子结构，并建议改进角度。

### 3. 痛点挖掘
扫描 Twitter 回复和讨论中关于 WordPress 性能、Shopify CRO、开发瓶颈的反复出现的挫败感。

### 4. 内容机会映射
生成可操作的内容创意：推文创意、逆向观点、权威建设帖子串、潜在客户磁铁概念。

### 5. 潜在客户情报
通过分析个人简介、招聘信号、增长阶段指标和网站链接来识别潜在客户。
