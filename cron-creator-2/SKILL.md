# 定时任务创建器

从自然语言请求自动创建 Clawdbot 定时任务。

## 快速安装（一条命令）

在终端中运行：

```bash
bash -c "$(curl -sL https://raw.githubusercontent.com/digitaladaption/cron-creator/main/install.sh)"
```

或手动安装：

```bash
# 安装技能
mkdir -p ~/.clawdbot/skills
git clone https://github.com/digitaladaption/cron-creator.git ~/.clawdbot/skills/cron-creator

# 配置并重启
clawdbot gateway restart
```

就这些！然后只需这样说：

- "每天早上8:45创建 Ikigai 提醒"
- "每2小时提醒我喝水"
- "设置周一上午9点的每周签到"

## 功能说明

1. **听取**你创建定时任务的请求
2. **解析**时间、频率、频道和消息
3. **自动创建**定时任务
4. **确认**完成

## 触发模式

这样说：
- "为...创建定时任务"
- "设置提醒..."
- "安排..."
- "提醒我..."
- "创建每日/每周签到..."
- "添加重复..."

## 示例

| 你说 | 会发生什么 |
|---------|-------------|
| "每天早上8:45创建 Ikigai 提醒" | 创建每天早上8:45的 Ikigai 日记提示 |
| "每2小时提醒我喝水" | 创建每小时喝水提醒 |
| "设置周一上午9点的每周签到" | 创建周一上午9点的每周回顾 |
| "每天早上7点叫醒我" | 创建每天早上7点的闹钟/提醒 |
| "每天早上6:30给我发送一句名言" | 创建每天早上6:30的名言 |

## 支持的时间格式

| 你说 | Cron 表达式 |
|---------|------|
| "上午8点" | `0 8 * * *` |
| "上午8:45" | `45 8 * * *` |
| "晚上9点" | `0 21 * * *` |
| "中午" | `0 12 * * *` |
| "午夜" | `0 0 * * *` |

## 支持的频率

| 你说 | Cron 表达式 |
|---------|------|
| "每天" / "每日" | 指定时间的每天 |
| "工作日" | 周一至周五的指定时间 |
| "周一" / "每周一" | 每周一 |
| "每小时" / "每小时" | 每小时的 :00 |
| "每2小时" | `0 */2 * * *` |
| "每周" | 每周（默认为周一） |
| "每月" | 每月（每月1日） |

## 频道

只需在请求中提及频道：
- "在 WhatsApp 上" → WhatsApp
- "在 Telegram 上" → Telegram
- "在 Slack 上" → Slack
- "在 Discord 上" → Discord

默认：WhatsApp

## 默认消息

技能自动生成适当的消息：

| 类型 | 默认消息 |
|------|-----------------|
| Ikigai | 包含目的、食物、运动、连接、感恩的晨间日记 |
