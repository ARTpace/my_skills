### 产品描述
一键自动化文件整理工具，智能分类、重命名、清理重复文件，让电脑井井有条。

### 核心功能
1. **智能分类** - 自动按文件类型分类（图片、文档、视频等）
2. **智能重命名** - 基于日期和内容自动命名
3. **重复检测** - 智能识别重复文件
4. **安全备份** - 操作前自动备份，支持撤销
5. **预览模式** - 先预览再操作，零风险
6. **跨平台** - 支持Windows/macOS/Linux

### 技术规格
- **处理速度**: 4500+ 文件/秒
- **内存占用**: < 100MB
- **系统要求**: Python 3.7+
- **文件大小**: 80KB (压缩包)
